# WSLabelme Skills 技能体系

本项目为 AI 图像标注与训练系统（基于 Labelme 重构）建立了完整的技能体系，用于指导开发、测试和维护工作。

## 📁 技能目录结构

```
.claude/skills/
├── README.md                    # 本文件，技能体系说明
├── 01-core-skills.md            # 核心开发技能
├── 02-ui-skills.md              # UI/界面开发技能
├── 03-debug-skills.md           # 调试与异常处理技能
├── 04-build-skills.md           # 构建与发布技能
├── 05-test-skills.md            # 测试技能
└── 06-maintenance-skills.md     # 维护与优化技能
```

## 🎯 技能分类

### 1. 核心开发技能 (01-core-skills.md)
- 项目架构理解
- PyQt5 开发规范
- 标注功能开发
- AI 集成开发
- 数据管理

### 2. UI/界面开发技能 (02-ui-skills.md)
- Qt Designer UI 文件使用
- 自定义 Widget 开发
- 样式与主题
- 响应式布局
- 图标与资源管理

### 3. 调试与异常处理技能 (03-debug-skills.md)
- 日志系统使用 (loguru)
- 异常捕获与分析
- 内存泄漏排查
- 性能问题分析
- 常见错误解决方案

### 4. 构建与发布技能 (04-build-skills.md)
- PyInstaller 打包
- 版本管理
- 配置文件管理
- 依赖管理
- 发布流程

### 5. 测试技能 (05-test-skills.md)
- 单元测试编写
- UI 自动化测试
- 功能测试
- 回归测试
- 性能测试

### 6. 维护与优化技能 (06-maintenance-skills.md)
- 代码重构
- 性能优化
- 安全问题处理
- 文档维护
- 问题追踪

## 📋 技能等级定义

| 等级 | 说明 | 要求 |
|------|------|------|
| L1 - 基础 | 了解基本概念 | 能够完成简单任务 |
| L2 - 熟练 | 掌握核心技能 | 独立完成常规任务 |
| L3 - 精通 | 深入理解原理 | 解决复杂问题，指导他人 |
| L4 - 专家 | 创新与优化 | 架构设计，技术决策 |
